# Muffin Clicker

A delightful incremental clicker game where you bake muffins, buy upgrades, and build your muffin empire!



## Features

- **Click to Bake**: Click the giant muffin to start your baking journey.
- **Upgrades**: Purchase various upgrades like "Grandma", "Muffin Stand", "Bakery", and more to automate your production.
- **Visual Progression**: Watch your bakery grow with visual representations of your upgrades.
- **Save System**: Your progress is automatically saved so you can continue baking later.
- **Responsive Design**: Looks great on desktop and mobile.

## How to Play

1.  **Click the Muffin**: Generate muffins by clicking.
2.  **Buy Upgrades**: Use your muffins to buy upgrades from the store on the right.
3.  **Increase MPS**: Upgrades increase your "Muffins Per Second" (MPS).
4.  **Unlock Achievements**: (Coming soon!)

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (ES6+, LocalStorage)

## Deployment

This website is ready to be deployed on Cloudflare Pages or GitHub Pages.

### Cloudflare Pages

1.  Connect your GitHub repository to Cloudflare Pages.
2.  Select the repository `muffin`.
3.  Set the **Build command** to `(none)`.
4.  Set the **Build output directory** to `(root)` or leave blank.
5.  Click **Save and Deploy**.
